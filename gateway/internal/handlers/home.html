ASFP-Pro Control Center

Центральный портал стенда: здесь можно проверить доступность сервисов, открыть документацию и перейти к ключевым инструментам (CRM, WMS, analytics, MinIO). Используйте HTTPS-коннект через nginx — сертификаты генерируются автоматически командой make up.

API Gateway

Точка входа для клиентов и интеграций: health-check, загрузка файлов, проксирование CRM/WMS. Доступно описание контрактов и smoke-сценарии.

Быстрые действия

  • Проверка стенда: make up ➝ make smoke ➝ make down.
  • Smoke с HTTPS: mkcert -install, затем SMOKE_GATEWAY_HTTPS_URL=https://localhost:8443 make smoke.
  • Документация: полный README и архитектурные описания находятся в репозитории repo/README.md.
